# MIPSim

A simple MIPS simulator, designed for Project Uranus.

## Current Progress

- [x] MIPS I instructions
- [x] Bus & IO support
- [x] virtual devices: RAM, ROM, GPIO, VGA
- [ ] Exceptions & interruptions
- [ ] MIPS 32 instructions
- [ ] JIT
